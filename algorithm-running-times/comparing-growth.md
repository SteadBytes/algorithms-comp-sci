# Comparing Function Growth
In order from slowest growing to fastest growing:
* Constant -> 1
* Logarithmic -> *log<sub>n</sub>* 
* Linearithmic -> *n &middot; log<sub>n</sub>*
* Polynomial -> *n<sup>2</sup>*, *n<sup>3</sup>* etc
* Exponential -> *2<sup>n</sup>*
* Factorial -> *n!*
